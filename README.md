# 일식 달그림자 속도 시뮬레이션 ☀️🌑🌍

지구 **자전 각속도**가 달 **공전 각속도**보다 약 27배 크지만, 일식 때 달그림자는 지표를
빠르게 지나가고 태양은 **오른쪽(서쪽)부터** 가려집니다. 왜 그런지를 **각속도 vs 선속도**
관점에서 4개의 시점으로 보여주는 순수 HTML/Canvas 시뮬레이션입니다. (빌드 도구·외부 라이브러리 없음)

> **라이브 데모:** Pages 활성화 후 → `https://devrobertson.github.io/solar-eclipse-speed/`

## 무엇을 보여주나

학생들이 흔히 하는 오해를 바로잡는 것이 목표입니다.

- **학생 주장(틀림):** “지구 자전 각속도(≈15°/h)가 달 공전 각속도(≈0.5°/h)보다 27배 크니까,
  관측자가 달을 앞질러 달이 상대적으로 뒤로 밀린다 → 태양을 **왼쪽부터** 가린다.” → *각속도만 비교한 오류*
- **실제(맞음):** 가려지는 방향을 정하는 것은 각속도가 아니라 **지표에서의 선속도**.
  달은 지구 중심에서 약 60배 멀리 있어, 각속도가 1/27이어도 선속도는 약 **2.2배** 더 크다.
  → 달그림자가 지면보다 동쪽으로 더 빨리 달려 태양을 **오른쪽(서쪽)부터** 가린다.

### 핵심 수치 (실제 값 기준)

| | 각속도 ω | 선속도 v = ω·r |
|---|---|---|
| 지구 자전(지표) | 큼 (≈15°/h) | ω·R<sub>지구</sub> ≈ **465 m/s** (≈1,673 km/h) |
| 달 공전 | 작음 (≈0.5°/h) | ω·d<sub>달</sub> ≈ **1,023 m/s** (≈3,684 km/h) |
| 비교 | 지구가 **27배** 큼 | 달이 **2.2배** 큼 |

```
v_달 / v_지표 = (ω_달 / ω_지구) × (d_달 / R_지구) = (1/27.4) × 60.3 ≈ 2.2
지면 기준 그림자 속도 ≈ 1,023 − 465 ≈ 559 m/s (≈2,011 km/h, 서→동)
```

상수: R<sub>지구</sub>=6,371 km, d<sub>달</sub>=384,400 km, 항성일 23.934 h, 항성월 27.32 일.

## 4개의 시점

1. **북극 상공 시점** — 일식 약 2시간 동안 지구는 30°만 자전, 달은 ~1°만 공전(둘 다 반시계). 각속도는 지구가 크지만 그림자(달)의 선속도가 2.2배라 관성계에서 관측자를 추월. *(달 거리는 압축)*
2. **관측자 하늘** — 달이 태양을 오른쪽(서)부터 가리는 모습. 학생들의 틀린 예측(왼쪽부터)을 점선으로 비교 가능.
3. **지표 위 경주(관성계)** — 달그림자(1,023 m/s)가 지표(465 m/s)를 추월하는 선속도 경주.
4. **막대그래프** — 각속도는 지구가, 선속도는 달이 크다는 것을 숫자로.

**컨트롤:** 재생/일시정지 · 시간 슬라이더 · 속도(0.25–4×) · “여러 시점” 스냅샷(1차 접촉 → 최대식 → 종료) ·
학생 예측 토글 · 벡터 토글.

## GitHub Pages로 배포하기

이 저장소에는 `.github/workflows/pages.yml`(GitHub Actions 자동 배포)이 포함되어 있습니다.
**한 번만** 저장소 설정에서 Pages를 켜면 됩니다.

**방법 A — GitHub Actions (권장)**
1. 저장소 **Settings → Pages** 이동
2. **Build and deployment → Source** 를 **GitHub Actions** 로 선택
3. 이 브랜치(또는 `main`)에 푸시되면 워크플로가 자동 배포 → 출력된 URL로 접속

**방법 B — 브랜치에서 직접 서빙**
1. 이 변경을 `main` 브랜치로 머지
2. **Settings → Pages → Source: Deploy from a branch** 선택, 브랜치 `main` / 폴더 `/ (root)` 지정

> 배포 URL은 보통 `https://<사용자명>.github.io/solar-eclipse-speed/` 형식입니다.

## 가정 & 한계

- 적도·정오 근처, 평행한 햇빛(태양 무한 원거리), 항성일/항성월 기준의 **단순화 모형**입니다.
- 화면의 거리·크기는 보기 좋게 과장했지만, **각속도·선속도의 비율은 실제 값**을 사용했습니다.
- 방향(좌·우)은 **북반구에서 남쪽 하늘을 볼 때**(동=왼쪽, 서=오른쪽) 기준입니다.
- 달 궤도 경사·이심률, 위도에 따른 그림자 속도 변화 등은 의도적으로 생략했습니다.
